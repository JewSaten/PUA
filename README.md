# PUA (Plugin Upload App)

VSCode 插件 - 应用分发工具

## 插件简介

PUA (Plugin Upload App) 插件旨在简化应用的构建与分发流程。通过与蒲公英平台的集成，帮助开发者快速将应用部署到分发平台进行测试和分享。

传统的应用分发流程需要手动敲命令构建打包，然后打开蒲公英后台进行上传，操作繁琐且效率低下。PUA 插件彻底改变了这种方式，将所有操作整合到 VSCode插件内部：

- **解放双手**：无需手动执行构建命令，也无需操作蒲公英后台
- **提高效率**：一键完成构建和上传，减少重复性工作
- **简化操作**：更直观的可视化操作配置界面
- **安全可靠**：在 VSCode 安全存储平台 API 凭证

## 核心功能

- **构建与上传**：支持自动构建 Flutter 应用并上传至蒲公英平台（适用Flutter项目）
- **快速上传**：直接上传已构建的 APK/IPA 文件
- **可视化配置面板**：提供直观的构建配置和分发平台配置界面

## 快速开始

- **快捷键**：使用插件预设的快捷键(Cmd+Shift+R)快速呼出
- **命令面板**：通过 `Cmd+Shift+P` 输入 "pua"
- **插件视图**：在 VSCode 侧边栏打开 PUA 视图

## API 资源

- 官方文档：https://www.pgyer.com/doc/view/api#paramInfo
- API 凭证获取：登录蒲公英后台的[API信息页面](https://www.pgyer.com/account/api)
